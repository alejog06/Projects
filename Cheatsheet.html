Common Css properties

Common Css

Properties Name What does it do? Example
<color> Text color It chooses a color of your desire for the text. color: blue;
<text-align> Text Align It aligns the texts to the center, start, end and justify it. text-align center; or end; or start;
<font-familyt> Family of the Font It chooses a font for the text. font-family: helvetica;
<font-size> Size of the Text It applies a size for the text based on pixels. font-size: 18px or 1.4em
<font-weight> Level of boldness It defines how bold the text or how thin it is. font-weight: bold; or thin; or thinner; or italic;
<background-color> Backgrounds Color It applies a color for the background of the text background-color: cian;
<background-image> Background Image It applies an image you linked from your files or the web using an URL. background-image: url(/resources/images/wood.jpeg);
<height> Element height Sets the height of an element, wether it's and image or text. height: 20px; or 5em;
<width> Element Width Sets the Width of an Element. width: 25px; or 7em;
<opacity> Element clarity It defines how clear is an element in a scale from 0-1. opacity: 0.3;
